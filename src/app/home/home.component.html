Purpose

This site is meant to be used by people taking part in organized bike rides. Ride leaders can create cue sheets, which the participants use on their phones, mounted to the handlebar. There is also a map that shows you where all the others on the ride (or in your group) are.

Live Site

You are currently on the live site. If you create a cue sheet here, it will remain till you or somebody else deletes it. Any changes you make to existing cue sheets will be saved. If you would like to try out the site, and look at ready-made cue sheets, you may visit the demo version of the site at RidePlanner Demo

Demo Site

This is the demo version of the RidePlanner site. Cue sheets you create here, or changes you make to existing cue sheets, will not be saved once you have closed the browser window. The live site is currently only available to OutCycling members.
